Gerenciamento de estado

Hook centralizado: useViagens
useViagens() → lê/salva no localStorage, fallback para seed data
Retorna: { viagens, addViagem, updateViagem, deleteViagem }
Usado em: TripsGrid, TripDetailPage, AdminPage

Isso cobre tudo do site atual com melhor organização. Sem estado global externo (Redux/Zustand) — useViagens via props é suficiente para esse tamanho de app.
